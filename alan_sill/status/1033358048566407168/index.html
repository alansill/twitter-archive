OpenSSH 7.8 finally implements new format by default for private key encoding. Seriously, upgrade your keys if you use RSA keys using "ssh-keygen -p -o -f (KEYFILE)" (won't change your public key, only private one) and you might as well pick a stronger passphrase along the way!
